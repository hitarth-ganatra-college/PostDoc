# PostDoc

Greenfield scaffold for the AI LinkedIn Post Generator.

## Requirements
- Node.js 20+
- npm 10+

## Setup
```bash
npm install
npm run dev
```

Copy environment variables:
```bash
cp .env.example .env.local
```

## Scripts
- `npm run dev` — start local dev server
- `npm run lint` — run lint checks
- `npm run build` — create production build
- `npm run start` — run production server

## Initial Modules
- `src/auth`
- `src/linkedin`
- `src/research`
- `src/generation`
- `src/image`
- `src/db`
- `src/session`
